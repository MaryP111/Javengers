Ομάδα 1: Website

- Εμφάνιση των καταχωρήσεων για κάθε προϊον, ίσως ταξινομημένα ως προς την τιμή και με επιλογή εμφάνισης στο χάρτη
- Φιλτράρισμα των καταχωρήσεων με βάση την απόσταση
- Homepage και συνολικά η εικόνα της σελίδας στο Front-end
- Το τμήμα της επικοινωνίας να γίνει λειτουργικό
- Ανανέωση και Διαγραφή προϊόντων

Ομάδα 2: Admin

- Προσθήκη ρόλου διαχειριστή στη σελίδα
- Δημιουργία dashboard για τον διαχειριστή
- Δυνατότητα επεξεργασία της βάσης σε φιλική μορφή (Πινακες)

Ομάδα 3: Rest API- Testing

- Δημιουργία του Rest API
- Έλεγχος των test cases
